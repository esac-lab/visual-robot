# visual-robot
Building visual robot with ORB_SLAM2, ZED and Turtlebot2 on Ubuntu melodic.

## System require

- Ubuntu 18
- x86_64
- Nvidia GPU

## How to use
Clone this repo and run `make_robot.sh`:

    git clone https://github.com/gaunthan/visual-robot.git
    cd visual-robot
    ./make_robot.sh
